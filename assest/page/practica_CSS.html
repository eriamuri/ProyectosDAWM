Hipervinculos

Hipervinculos: abrir en una nueva pestaña

Un hipervínculo es un enlace que lleva a otro recurso, como otra página web o un archivo. Para abrir un hipervínculo en una nueva pestaña, se usa el atributo target="_blank" en el código HTML de la etiqueta a, o se puede hacer manualmente manteniendo presionada la tecla Ctrl (o Cmd en Mac) al hacer clic en el enlace en muchos navegadores. abrir en una nueva pestaña


Hipervinculos:abrir en la misma pestaña

Un hipervínculo que se abre en la misma pestaña, usando el valor _self, redirige al usuario dentro de la misma ventana o marco sin abrir una nueva, lo que se considera el comportamiento predeterminado del navegador para enlaces internos. Esto es útil para navegar por secciones del mismo documento o sitio web, pero debe evitarse para enlaces externos o al salir de un proceso activo (como un formulario), ya que puede hacer que el usuario pierda su progreso actual. abrir en la misma pestaña


Hipervinculos: abrir una nueva pestaña, modo seguro

noopener noreferrer es un conjunto de atributos HTML que, cuando se usan en un enlace (a href="..." target="_blank" rel="noopener noreferrer">), protegen contra vulnerabilidades de seguridad y protegen la privacidad del usuario. noopener evita que la página de destino acceda a la ventana original de tu sitio web, protegiendo de ataques de phishing y mejorando el rendimiento. noreferrer impide que la página de destino sepa de dónde provino el enlace, ocultando la información de referencia. abrir en una misma pestaña (modo seguro)
